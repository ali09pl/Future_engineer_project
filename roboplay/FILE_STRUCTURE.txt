🚀 كيفية البدء:

1. انتقل إلى المشروع:
   cd /home/ubuntu/roboplay-platform

2. ثبت المتطلبات:
   pnpm install

3. ابدأ خادم التطوير:
   pnpm dev

4. افتح المتصفح:
   http://localhost:5173
